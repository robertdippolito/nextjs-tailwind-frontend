In this file we will describe the steps required to host our website. These steps will be executed during part 2 of the website from scratch series. 

1. Purchase a domain name 
2. Register a hosted zone in Route 53
3. Request a certificate from ACM
4. Create an S3 bucket and upload your static files
4. Create a CloudFront distribution and set the origin to your S3 bucket
5. Create your CNAME/A Records in Route53 for your domain
